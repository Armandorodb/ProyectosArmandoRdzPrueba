$\hat{y}$ = $\beta0$ + $\beta1X$

In[33]:
#Calculamos el promedio de X
xBar = np.mean(x)

#Calculamos el promedio de y
yBar = np.mean(y)

#Es el numerador de B1 para evitar errores
B1num = sum((x-xBar)*(y-yBar))

#Es el denominador de B1 para evitar errores
B1den = sum((x-xBar)**2)

# Calcula B1 como la proporcin entre el numerador y el denominador
B1 = B1num/B1den

# Calcula B0
B0 = yBar-(B1*xBar)

# Imprime el valor de B0

print("B0 = ", B0 )

Se calcula el error estndar (SE) de $\beta1$, el cual es de utilidad para juzgar la confianza de $\beta1$ y ser crucial para encontrar su intervalo de confianza.

In[36]:
# Se obtiene el numero de muestras
n = len(x)

SEB1 = np.sqrt(RSS/((n-2)*sum((x-xBar)**2)))

SEB1 = 0.09983378435340727

Podemos decir con un 95% de seguridad que por cada aumento unitario en el logaritmo base 10 del GDP la felicidad aumenta entre 0.4307 y 0.8255 puntos.

Ahora se determinar si existe alguna asociacin entre la felicidad y el logaritmo base 10 del GDP, para esto realizaremos una prueba de hiptesis:

Hiptesis nula (H0): $\beta1$ = 0

Hiptesis alternativa (H1): $\beta1$ $\neq$ 0

Para poder determinar esto debemos de obtener el estadstico t, para posteriormente obtener el valor de p, para una funcin t de student con n-2 grados de libertad.

Una vez tengamos estos valores vamos a comparar el valor p con el de alpha en este caso de 0.05, lo que implica que tiene una certeza del 95%, si el valor de p es menor podemos decir que se rechaza la hiptesis nula y podemos presumir que si existe una relacin entre la felicidad y el GDP.

In[38]:
# Calcula el estadstico t
t = B1 / SEB1
# Calcula el p-value asociado
p = st.t.sf(abs(t),n-2)*2
# Muestra en consola el resultado del estadstico t
print ( "Valor de t-statisic=",t)
# Muestra en consola el resultado del p-value
print ("Valor de p-value=",p, "\n")

if p<0.05:
    print("Se rechaza la hiptesis nula, se tiene una confianza del 95% de que existe una relacin entre la felicidad y el GDP")
else:
    print("No se rechaza la hiptesis nula, puede no existir relacin entre la felicidad y el GDP")
